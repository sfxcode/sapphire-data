    Sapphire Data

    A Framework for data handling. It provides an adapter for beans and maps with JavaFX Property support.

    Sapphire Frameworks

    Sapphire Data is heavily used in the sapphire ui frameworks:

    • sapphire-core
    • sapphire-extension

    Cross Build

    Build and tested against Scala 2.12/2.13 and JDK 11 - 14

    Framework Dependencies

    JavaFX

    Java UI Application Framework as replacement for Swing.

    Sapphire depends on OpenJFX base package for properties handling.

    https://openjfx.io

    Expression Language

    Expressions are resolved by EL 3 Tomcat Expression Language.

    Maven

    Sapphire is published to Bintray and linked to Maven Central.

    Repository

    resolvers += "sfxcode-bintray" at "https://dl.bintray.com/sfxcode/maven"

    Artifact

    sbt
    libraryDependencies += "com.sfxcode.sapphire" % "sapphire-data_2.12" % "1.0.2-SNAPSHOT"
    Maven
    <dependency>
  <groupId>com.sfxcode.sapphire</groupId>
  <artifactId>sapphire-data_2.12</artifactId>
  <version>1.0.2-SNAPSHOT</version>
</dependency>
    Gradle
    dependencies {
  compile group: 'com.sfxcode.sapphire', name: 'sapphire-data_2.12', version: '1.0.2-SNAPSHOT'
}
